# URL Shortener (React + Vite + Spring Boot)

## Overview
A full-stack URL shortener with JWT authentication, Redis caching, and analytics.

## Features
- Register/Login with JWT
- Shorten URLs
- Redirect short URLs
- View hit count and creation date
- Backend: Spring Boot + Postgres + Redis
- Frontend: React + Vite
- Deployment: Render (backend) + Vercel (frontend)

## Live Demo
- Frontend: https://urlshortener-frontend-black.vercel.app/
- Backend: https://url-shortener-x6c5.onrender.com
